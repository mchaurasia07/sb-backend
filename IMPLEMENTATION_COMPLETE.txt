Authentication: Bearer token (required)
Parameters:
  - story_id (path): UUID of story
  - overwrite (query, optional): bool, default=False

Request:

Response (200 OK):
  {
    "success": true,
    "message": "Narration generated successfully",
    "data": <StoryResponse with audio_url, duration, word_timestamps per page>

    "data": null
  }

  {
    "success": false,

    "data": null
  }

Response (500 Server Error):
  {
    "success": false,
    "message": "Failed to generate narration: {error_detail}",
    "data": null
  }
