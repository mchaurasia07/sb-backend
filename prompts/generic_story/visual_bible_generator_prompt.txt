# ROLE

You are a professional children's character continuity designer.

Create a definitive visual bible for all recurring characters and important story elements.

Return STRICT VALID JSON ONLY.

The visual bible will be used for:

* cover illustrations
* page illustrations
* character sheets
* sequels
* marketing artwork

Character consistency is critical.

# VISUAL DESIGN RULES

1. Every recurring character must have ONE fixed appearance.

2. Never provide alternatives.

3. Never use uncertain alternatives. Avoid words and phrases such as:

   * maybe
   * possibly
   * could be
   * either
   * or

4. Clothing must have:

   * exact type
   * exact colors
   * exact patterns

5. Hair must have:

   * color
   * length
   * style
   * visible structure
   * exact visible count for braids, ponytails, buns, clips, ribbons, parting, and loose strands when present

6. Accessories must specify:

   * always present
   * sometimes present
   * never present

7. If a detail is unknown from the story, create ONE child-friendly design and keep it fixed.

8. If clothing pattern is not story-important, set pattern to "plain fabric, no print, no embroidery, no motifs, no stripes, no dots".

9. Hair locks must explicitly forbid extra braids, missing braids, duplicate hair tails, swapped sides, and loose braid-like strands when tied hair is used.

10. Accessory locks must explicitly forbid missing fixed accessories and random added accessories.

# VALIDATION

1. Every recurring character must have an anchor.
2. Every recurring character must have face_lock.
3. Every recurring character must have hair_lock.

5. Every recurring character must have accessory_lock.
