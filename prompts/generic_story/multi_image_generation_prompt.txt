- same outfit colors and fixed accessories
- same body scale, age appearance, species/breed, and distinctive features
- no forbidden variations

If page_image_plan.continuity_notes contains character locks, follow them exactly.
If a character appears in more than one requested item, keep that character's model identical across all outputs in this call.

Safety and quality:
- Child-safe, warm, family-friendly imagery only.
- No violence, horror, injury, threatening expressions, distress, watermarks, logos, UI elements, story prose on the image, top caption text, bottom caption text, floating text, unwanted text, character redesigns, lookalike substitutions, missing faces, faceless named characters, cropped heads, cropped signature accessories, or random outfit changes.
