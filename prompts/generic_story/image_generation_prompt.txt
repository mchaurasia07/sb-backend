# CONTRACT

If PAGE TYPE is "cover", create a finished front book cover from the whole-story direction, title layout, genre signal, and story promise. It must not look like page 1 or any interior page. Visible cover characters must match the same canonical character models used for all interior story pages.

If PAGE TYPE is an interior story page, draw the exact SCENE TO DRAW and IMAGE PLAN moment: action, environment, character list, important objects, object states, emotion, camera shot, and composition are hard requirements.

Use only characters and story-important objects listed in the image plan. Do not add extra characters, family members, toys, props, locations, text, signs, labels, logos, or background story elements unless explicitly listed. Natural incidental background details are allowed only when they are required by the listed environment and do not become new story-important objects.

Do not carry a recurring/signature object into this page unless it is listed in IMAGE PLAN.important_objects for this page. Objects from earlier or later pages must be omitted when the current page scene does not call for them.

Keep identities and continuity from the model sheet and GLOBAL CHARACTER REFERENCE JSON. Do not redesign characters or change outfit, hairstyle, face shape, age, skin tone, body scale, accessories, fixed objects, or forbidden variations. For covers, use the exact same story characters as the interior pages, not marketing redesigns, alternate costumes, older/younger versions, prettier versions, simplified versions, or lookalikes.

Important object consistency is mandatory for objects listed in IMAGE PLAN.important_objects on this page. Every listed object must keep the same locked identity from the model sheet: same material, shape, color, size, markings, and normal orientation. Only change the object's state when IMAGE PLAN.object_states explicitly says it changes on this page.

Before drawing, silently check IMAGE PLAN.object_states. Draw each listed object's exact current state for this page: position, orientation, open/closed state, empty/full state, water level, amount/count of contents, damage, cleanliness, lighting state, or other visible story state. Do not preview a future solved state and do not revert to an earlier state.

For cause-and-effect object sequences, preserve the timeline. A pot with low water must remain low until pebbles are added; the water should rise gradually only on the pages where the plan says it rises. Similar story mechanisms must remain visually sequential and plausible.

Character consistency is mandatory for every visible character, not only the hero.

Respect basic scene etiquette. If the scene is inside a temple, prayer room, sacred space, or a home area where shoes are removed, do not draw outdoor shoes on feet; show bare feet, socks, or footwear placed neatly near the entrance. If a character is on a bed, mattress, sofa used for sleeping, or bedding, do not draw outdoor shoes on the bed or on the character's feet. This does not change the locked footwear design; it only controls whether footwear is worn in that scene.

Before drawing, silently check every character listed in IMAGE PLAN.characters against the CHARACTER AND CONTINUITY MODEL SHEET:
- same face and facial proportions
- same hair/fur style and color
- same skin/body color
- same outfit colors and fixed accessories
- same body scale, age appearance, species/breed, and distinctive features
- no forbidden variations

If IMAGE PLAN.continuity_notes contains character locks, follow them exactly.

Side characters, helpers, parents, grandparents, friends, classmates, animals, and antagonists must stay visually consistent across pages.

For covers, character consistency is stricter than cover styling: title layout, cover composition, and decorative framing may change, but face, hair, clothing, footwear, age/body scale, skin tone, and fixed accessories must not change from GLOBAL CHARACTER REFERENCE JSON.

Do not replace a named character with a lookalike, generic child, generic adult, generic animal, or background figure.

Named characters important to the scene should have identifiable faces. Do not hide, blur, crop off, or omit faces unless the image plan explicitly calls for a back view or closeup.

Do not crop off heads, hands, feet, tails, wings, fixed accessories, or signature objects for visible named characters.

Draw each named character listed in IMAGE PLAN.characters exactly once. Do not draw a second head, second face, second body, reflection, portrait, photo, shadow-person, or lookalike version of the same character.

Every visible head must be attached to its correct body. No floating heads, detached heads, disembodied faces, partial duplicate heads, extra child heads, or head-only background cutouts.

Each interior page must look like its own story moment. Do not reuse generic poses, nursery/mat setups, object layouts, or previous-page compositions unless requested.

Follow TITLE AND TEXT INSTRUCTION exactly.
For interior story pages, never render story prose, narration, captions, subtitles, title text, floating text, UI text, top text, bottom text, or overlay text.
Interior story pages may contain readable words only when IMAGE PLAN explicitly requires short in-scene object text, such as blackboard writing, a chair name label, a classroom sign, or a book cover inside the scene. Keep that text physically on the object. Do not place any words in empty sky/wall/negative space as a caption.
For covers, the only allowed text is the exact cover title.

Composition: one clear focal point, child-friendly clean layout, expressive faces, no clutter. For interior story pages, use one unified full-page illustration, not a split scene, comic panel layout, montage, collage, or multiple vignettes.

Quality bar: crisp focal subject, clean silhouettes, detailed but uncluttered environment, polished light and shadow, appealing proportions, professional children's picture-book finish.

Negative rules: no scary elements, violence, distorted hands, extra fingers, missing limbs, missing faces, faceless named characters, cropped heads, duplicate heads, duplicate faces, detached heads, disembodied faces, partial duplicate bodies, cropped signature accessories, watermark, random outfit changes, character redesigns, lookalike substitutions, story prose on the image, top caption text, bottom caption text, or unwanted text.

# OUTPUT

Return only the finished illustration.
